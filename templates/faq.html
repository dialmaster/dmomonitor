Initial Setup:

  1. If you don't already have it, download the official Dynamo Miner
  2. Create an account on this site. Once you do you will get a unique Cloud Key, which you can find in My Account
  3. Download DMO-Wrapminer. You will use this to run your miners locally, allowing them to connect to DMO-Monitor
  4. Extract the contents of the DMO-Wrapminer archive into the same directory where your DynMiner executable is.
  5. Edit the provided dmowrapconfig.yaml file, putting your information and desired configuration in.
  6. Make sure you put the Cloud Key you got from My Account into dmowrapconfig.yaml
  7. Run dmo-wrapminer, passing the yaml config as a command line option like: dmo-wrapminer.exe dmowrapconfig.yaml

You should now be able to visit Mining Monitor and see your runnning miner statistics!

Telegram Notifications:

In order to receive realtime notifications from Telegram if your miners go down/offline do the following:

  1. Create a Telegram Account and download the app on your device.
  2. Send a message to userinfobot on Telegram to get your Telegram User ID. See instructions HERE
  3. Go to My Account and add your telegram user id from the previous step
  4. Now find dmo_monitor_bot on telegram and send the message /start to it.

Now test it out. Take a miner offline, wait about 1 minute, and see if you get a notification that your miner is offline.

Mining Statistics:

This step is simple. If you are solo mining, you can put your receiving address into My Account and you will see the statistics for coins mined to that address.

Note that you can put in any receiving address here and you will be able to see mined coin statistics for that address since it is public information on the blockchain.
