Once you create an account, you will need to run your miners with dmo-wrapminer to take advantage of the miner monitoring.

Linux binary package available HERE.
Windows binary package available HERE.

What it does:
  • Acts as secure connection between your miner and dmo-monitor.com.
  • Makes running your miner more reliable by auto-restarting if it dies.
  • Allows you to specify a time interval to restart your miner.
  • Manages your miner settings in an easy to edit config file.

How to use it:
  • Extract it into the same directory with your DynMiner OR SRBMiner executable
  • Update the .yaml configuration file that comes with it.
  • Run it, passing the name of the config you with to use: dmo-wrapminer myconfig.yaml
  • If you do not pass a config file name, then it will look for mydmowrapconfig.yaml as a default
  • To run a dmo-wrapminer that AUTOMATICALLY checks for updates and updates itself, please run dmo-launcher instead!

If you choose to use dmo-launcher then your systems will automatically be kept up to date with any future updates required for remote management (or any other new features that get added).

After the beta, this wrapper will be what allows you to remotely update your miner configurations, restart your miner, and deploy new versions of DynMiner directly from this site.
