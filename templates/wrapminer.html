Once you have created an account, you will need to run your miners using dmo-wrapminer in order to take advantage of the monitoring capabilities.

Linux binary available HERE.
Windows binary available HERE.

After you download this:
1) Extract it into the same directory with your DynMiner executable.
2) Update the .yaml configuration file that comes with it.
3) Run it:
dmo-wrapminer myconfig.yaml
This mining wrapper:
1) Acts as an intermediary between your miner and dmo-monitor.com.
2) Makes running your miner more reliable by auto- restarting if it dies.
3) Allows you to specify a time interval to restart your miner.
4) Manages your miner settings in an easy to edit config file.

In the future, this wrapper will also be what allows you to automatically make updates to your miner configurations, restart your miner remotely, and deploy new versions of DynMiner directly from this site.
