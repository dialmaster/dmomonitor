Once you create an account, you will need to run your miners with dmo-wrapminer to take advantage of the miner monitoring.

Linux binary available HERE.
Windows binary available HERE.

What it does:
  • Acts as an intermediary between your miner and dmo-monitor.com.
  • Makes running your miner more reliable by auto- restarting if it dies.
  • Allows you to specify a time interval to restart your miner.
  • Manages your miner settings in an easy to edit config file.

How to use it:
  • Extract it into the same directory with your DynMiner executable
  • Update the .yaml configuration file that comes with it.
  • Run it: dmo-wrapminer myconfig.yaml

After the beta, this wrapper will be what allows you to remotely update your miner configurations, restart your miner, and deploy new versions of DynMiner directly from this site.

Important Notes (beta):
 Dmo-wrapminer currently only allows for ONE running instance of the wrapper per computer. The MinerName is the unique key for what is displayed in dmo-monitor for each miner. Each instance of dmo-wrapminer you run must be setup with it's own unique MinerName in the yaml config.
