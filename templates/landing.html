The always free-to-use DMO Monitor application can be run as a local application and provides:
  • Monitoring of your miners and their hashrates
  • Notifications if a miner goes offline, via Telegram
  • Display of mining statistics such as coins per day, coins per hour
  • Native apps for both Windows and Linux
